Requires libusb-1
Requires hdf5-1.8

This pile of code is intended to read waveforms from oscilloscopes for analysis in computer.

It is known to work with Tektronix DPO2024 and TDS2024B oscilloscopes, from both Linux (2.6 kernel, 2011) and Mac OS X Lion (10.7), through a USB cable.

Do whatever you want with the code.
